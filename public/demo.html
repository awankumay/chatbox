Demo Aplikasi Chatbox

Ini adalah demonstrasi aplikasi chatbox sederhana yang dapat diembed ke situs web lain. Untuk mencoba, klik ikon chat di pojok kanan bawah.

Cara Penggunaan

Untuk menggunakan chatbox ini di situs web Anda sendiri, tambahkan kode berikut ke dalam halaman HTML Anda:

<script type="module">
  import ChatboxWeb from "http://localhost:3000/assets/js/web.js";
  ChatboxWeb.init({
    isEmbed: true,
    secretKey: "your-secret-key",
    baseURL: "http://localhost:3000",
    theme: {
      button: {
        backgroundColor: '#4CAF50',
        size: 'large',
        customIcon: 'https://github.githubassets.com/assets/GitHub-Mark-ea2971cee799.png'
      },
      chatWindow: {
        welcomeMessage: 'Halo! Ada yang bisa kami bantu?'
      }
    }
  });
</script>

Fitur

  • UI chatbox yang responsif
  • Konfigurasi tampilan yang fleksibel
  • Mudah diintegrasikan ke situs web lain
  • Support untuk kustomisasi tema
